## [1.0.0] - 2025-02-08

### Added
- Full UX redesign from single-file app to modular ES modules
- Dashboard view with workspace grid, sorting, starring, deletion
- Split-view reader (30% card list / 70% reader pane)
- Rich text editor with H1/H2/H3, bold, italic, lists, text color, highlight
- Heading outline tree with click-to-scroll navigation
- Search overlay with Ctrl+S/Ctrl+K keyboard shortcut
- Dark mode toggle with 8 theme presets
- Document parsing: PDF, DOCX, TXT, Markdown
- PDF and JSON export
- JSON import with category merging
- Category management with add/delete
- Workspace name inline editing
- localStorage-based persistence with Supabase migration abstraction
- View router (dashboard <-> workspace)
- Docker container support
- Vercel static deployment config
- GitHub Actions CI/CD pipeline
- Professional project structure (frontend/, assets/, docs/, docker/)
- Rebranded to KOGNIV
